# bonfire

## 持ち物

- 焚火台
- ナタ / ナイフ / ノコギリ
- 着火剤
- ライター
- 火箸

## 推奨グッズ

- ノコギリ
  - [シルキー ゴムボーイ 万能目 210mm](https://www.amazon.co.jp/dp/B000CEAX8S/)
    - より大型の製品もあるが、枝を切る程度であれば210mmで十分。
- ライター
  - [SOTO スライドガストーチ](https://www.amazon.co.jp/dp/B004WMPFWA/)
    - 先端が伸びるので奥底に埋まった着火剤などに点火するのが容易。
    - OD缶のガスがそのまま入れられるらしい。
